Masviejo

Utilidades para gestionar y descargar los datos almacenados en la base de datos local (SQLite).

💾 Descargar base de datos (wu.db) 📄 Exportar CSV 📊 Ver estadísticas

Cómo usar

  • Los datos se guardan automáticamente en la base de datos persistente.
  • Puedes descargar la base SQLite para analizarla o subirla a Rails.
  • También puedes exportar los registros como CSV.
